# ВЕКТРА — Система проверки документов

Веб-приложение для проверки конкурентных предложений и счетов на подлинность с помощью AI.

## Возможности

- **AI-анализ документов** — загрузка PDF и фотографий, автоматическая проверка по 17 критериям
- **Математическая верификация** — JavaScript перепроверяет все расчёты (НДС, суммы строк, итого)
- **Админ-панель** — управление пользователями, ролями и правами
- **3 роли**: Администратор, Руководитель, Менеджер
- **История проверок** — все результаты сохраняются в базе данных

## Быстрый деплой (30 минут)

### Шаг 1: Создать базу данных (Supabase — бесплатно)

1. Зайдите на **https://supabase.com** → Sign Up (через GitHub)
2. Нажмите **New Project** → введите название (например `vectra`)
3. **Запишите пароль** от базы данных
4. Дождитесь создания (1-2 минуты)
5. Перейдите в **SQL Editor** → нажмите **New Query**
6. Скопируйте содержимое файла `sql/schema.sql` → нажмите **Run**
7. Перейдите в **Settings → API** → скопируйте:
   - **Project URL** (это NEXT_PUBLIC_SUPABASE_URL)
   - **anon public** key (это NEXT_PUBLIC_SUPABASE_ANON_KEY)
   - **service_role** key (это SUPABASE_SERVICE_KEY) — ⚠️ секретный!

### Шаг 2: Получить ключ Anthropic API

1. Зайдите на **https://console.anthropic.com**
2. Создайте аккаунт, добавьте карту
3. Перейдите в **API Keys** → **Create Key**
4. Скопируйте ключ (начинается с `sk-ant-...`)

### Шаг 3: Деплой на Vercel (бесплатно)

**Вариант А — через GitHub (рекомендуется):**

1. Создайте репозиторий на **https://github.com** → загрузите туда эту папку
2. Зайдите на **https://vercel.com** → Sign Up через GitHub
3. Нажмите **Import Project** → выберите ваш репозиторий
4. В разделе **Environment Variables** добавьте:

```
NEXT_PUBLIC_SUPABASE_URL = https://xxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY = eyJ...
SUPABASE_SERVICE_KEY = eyJ...
ANTHROPIC_API_KEY = sk-ant-...
JWT_SECRET = любая-длинная-случайная-строка-минимум-32-символа
```

5. Нажмите **Deploy** — через 2 минуты сайт готов!

**Вариант Б — через CLI:**

```bash
npm install -g vercel
cd vectra
vercel login
vercel --prod
# Vercel попросит указать переменные окружения
```

### Шаг 4: Первый вход

1. Откройте ваш сайт (например `vectra.vercel.app`)
2. Войдите: **admin** / **admin123**
3. Зайдите в Пользователи → добавьте реальных пользователей
4. **Смените пароль админа!** (через ✏️ в списке пользователей)

## Подключение своего домена

1. В Vercel → Settings → Domains → добавьте `vectra.ваш-домен.ru`
2. У регистратора домена добавьте CNAME запись: `vectra` → `cname.vercel-dns.com`
3. SSL-сертификат создастся автоматически

## Стоимость

- **Vercel** — бесплатно (до 100 GB трафика/мес)
- **Supabase** — бесплатно (до 50 000 запросов/мес, 500 MB БД)
- **Anthropic API** — ~4-6 ₽ за проверку (~$0.05)
